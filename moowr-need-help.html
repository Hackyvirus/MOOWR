Steps to Use the Duty Saving Calculator - MOOWR Scheme

1. Visit the Index Page

  1. Open the Optitax’s website and find Duty Saving Calculator - MOOWR Scheme. You will land on the Index page, which provides an introduction to the Duty Saving Calculator - MOOWR Scheme and an overview of its features.

  2. Review the introduction and steps, and then click on the “Calculate Now” button.

2. Navigate to the Calculator Page

  1. Click on the “Calculate” button to be redirected to the Calculator page.

3. Accept Terms and Conditions

  1. A pop-up with the Terms and Conditions will appear when you first visit the page. Review and accept these terms to proceed.

4. Access the Main Calculator Page

On the Calculator page, You can see 2 links for Your Help, One is How to Use MOOWR Calculator and Second is FAQs page

  1. Gross Value Method: Enter details for Capital Goods and Raw Material, as well as Common Questions.

  2. Individual Value Method: Enter individual values as required.

Gross Data Calculation

1. Enter Capital Goods Gross Data

  1. Enter Value in Gross value of import (CIF)

  2. Enter Value in Gross value of BCD

  3. Enter Value in Gross value of AIDC (If applicable)

  4. Enter Value in Gross value of ADD (If applicable)

  5. Enter Value in Gross value of IGST

  6. Choose Value from Disposal of capital goods after end of useful life if you choose Sale in DTA then only you have to Enter Intended period of use (in Years)

  7. Enter Value in Safeguard Duty

  8. Enter Value in Countervailing Duty

  9. Click on Next Button

2. Enter Raw Materials Gross Data

  1. Enter Value in Gross value of import (CIF) in first year

  2. Enter Value in Gross value of BCD

  3. Enter Value in Gross value of AIDC (If applicable)

  4. Enter Value in Gross value of ADD (If applicable)

  5. Enter Value in Gross value of IGST

  6. Enter Value in Safeguard Duty

  7. Enter Value in Countervailing Duty

  8. Click on Next Button

3. Enter Common Questions Data

  1. Enter Value in Expected Project life cycle (in years)

  2. Enter Value in Expected Annual Growth (%)

  3. Enter Value in domestic Sales

  4. Enter Value in Export Sales

  5. Enter Value in Expected rate of interest on working capital (Default 9%):

  6. Enter Value in Time gap (in no. of days) for utilisation of GST (input tax credit)

  7. Enter Value in Gross annual value of RoDTEP

  8. Enter Value in Gross annual value of All Industry drawback

  9. Enter Value in Time taken for conversion of raw material (in no. of days)

  10. Click on Calculate Button

Individual Data Calculation

1. Enter Capital Goods Individual Data

  1. Choose BCD rate from the option, if you choose Others then you can add BCD rate manually.

  2. Enter Value in Import value (CIF)

  3. Enter rate of AIDC (if applicable)

  5. Enter Value in Safegourd Duty (if applicable)

  6. Enter Value in IGST Amount

  8. Enter Value in Intended period of Use, if you have choose "Sale in DTA" then only this option activate

  9. Click on Next Button
